## Sistema de Hospital

## Descripción

Este proyecto simula un sistema de pacientes en un hospital utilizando Programación Orientada a Objetos.
Se usa una clase abstracta `Paciente`, junto con clases `Medico` y `Seguro`, y una clase final `PacienteHospital` que aplica herencia múltiple y encapsulamiento.

## Ejecución

Para ejecutar el programa, abra el programa que esta dentro de la carpeta y lo ejecuta.
